Mapr - Business Management for Tradesmen

Mapr is an intuitive, all-in-one solution designed to streamline the workflow of tradesmen by providing a suite of tools for calculators, invoice management, and business operations. It aims to improve efficiency, reduce administrative overhead, and enhance productivity for trades professionals.

Key Features
	•	Calculators: Simplify calculations and estimates for projects, allowing tradesmen to work faster and more accurately.
	•	Invoice Management: Easily create, track, and manage invoices for completed jobs, saving time on administrative tasks.
	•	Business Management Tools: Organize and manage business operations, from client communication to job scheduling, all in one place.
	•	User-Friendly Interface: A clean, modern interface that makes it easy for users to navigate and access key features.

Technologies Used
	•	React: A JavaScript library for building user interfaces, ensuring a responsive and dynamic application.
	•	Styled-Components: A library for styling React components with CSS-in-JS for clean and modular styles.
	•	Smooth Scrolling: Implemented smooth scrolling for a better user experience, making the page feel more dynamic.
	•	Fixed Content: Key elements such as the app name, title, and description are fixed in position, providing a consistent header for the user while they scroll through mockups and images of the app in use.

Layout and Design

Hero Section

The landing page features a Hero Section with:
	•	Fixed App Name & Logo: The app name “Mapr” along with a logo that stays fixed while scrolling, making it always visible at the top of the page.
	•	Text Content: A bold title and description that clearly explain the value of the app, positioned fixed to the screen’s center, ensuring it’s always readable during the user’s scroll.
	•	Mockup Images: A series of images showcasing the app’s functionality, placed to the right of the text content. These images scroll smoothly, giving the user a dynamic and interactive experience.

Layout Details
	•	Background: A visually appealing gradient background, transitioning from a light green to a darker shade, designed to catch the user’s attention.
	•	Smooth Scrolling: The scrolling of mockup images and content is smooth, providing a polished experience without jerky movements.
	•	Responsive Design: The layout and styling are designed to adjust fluidly for different screen sizes, ensuring the app looks great on both desktop and mobile devices.
